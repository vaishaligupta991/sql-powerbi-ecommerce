## 🛒 Brazilian E-Commerce SQL Analysis

## 📌 Project Overview
SQL analysis of 100,000+ real Brazilian e-commerce orders using MySQL.

## 🛠️ Tools Used
- MySQL Workbench
- Dataset: Olist Brazilian E-Commerce (Kaggle)

## 📊 SQL Concepts Covered
- Aggregations & GROUP BY
- JOINS (multiple tables)
- CASE statements
- CTEs (Common Table Expressions)
- Window Functions (RANK, LAG, SUM OVER)
- Date functions (DATEDIFF, DATE_FORMAT, YEAR)

## 🔍 Key Insights
- Average delivery time: 12.5 days
- Revenue grew from 21K (2016) to 32L (2018)
- Credit card is the dominant payment method
- Seasonal revenue dips in April & June 2017
